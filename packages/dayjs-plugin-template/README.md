## install
> demo usage of this repo

```sh
yarn add @aiou/dayjs-plugin-template dayjs
```

## usage

### `Hello`

```ts
import Hello from '@aiou/dayjs-plugin-template'

dayjs.extend(Hello)
dayjs.hello().say()
```

params of `hello()`

- `word`
